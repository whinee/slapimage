    Important

    whinesnips is still in Development Stage.

    Using the library at this stage is not recommended.

    What’s this?

    whinesnips is a python library ― that is named to sound like "whine's nips" (nipples) ― that I made in order to separate and unify all of my python utility functions in a singular repository.

    Supported OSes

    Supposedly every operating system that supports running python, more preferably natively.
